# 🚀 VentureMind AI — Startup Intelligence Platform

VentureMind AI is an **AI-powered startup analytics platform** that predicts startup success using machine learning.

The system evaluates multiple startup factors such as funding, team size, market size, founder experience, and operational years to estimate the probability of success.

It also provides **startup valuation estimation, risk analysis, investor recommendations, and market opportunity insights** through an interactive dashboard.

---

# 🧠 Key Features

- 📊 Startup Success Prediction using Machine Learning  
- 📈 Startup Score (0–100) similar to a credit score  
- ⚠️ Risk Level Analysis  
- 💰 Startup Valuation Estimator  
- 🧑‍💼 Investor Recommendation System  
- 🌍 Market Opportunity Analyzer  
- 👨‍💻 Founder Strength Analysis  
- 📉 Feature Importance Visualization  
- 🧭 Startup Radar Profile  
- 📊 Startup Health Gauge  
- 📄 Downloadable Startup Analysis Report  
- 🎨 Interactive Streamlit Dashboard

# ⚙️ Installation

Clone the repository

git clone https://github.com/AryanDekate12/venturemind-ai-startup-intelligence.git

Go into the project folder

cd venturemind-ai-startup-intelligence

Install dependencies

pip install -r requirements.txt

---

# 📊 Generate Dataset

python generate_dataset.py

---

# 🤖 Train Machine Learning Model

python train_model.py

---

# 🚀 Run the Application

streamlit run app.py

The dashboard will automatically open in your browser.
